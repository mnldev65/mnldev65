<h2>01.  Banco Digital:</h2>

Aplicativo mobile de banco digital desenvolvido em **Flutter**, com autenticação de usuários via **Firebase** e interface moderna inspirada em aplicativos bancários reais.

**Principais funcionalidades:**
- 🔐 Cadastro e autenticação de usuários com Firebase Authentication

- 🧭 Navegação entre telas utilizando rotas do Flutter
- 📱 Interface responsiva seguindo o Material Design

**Stack técnica:**

- Firebase Authentication
- Cloud Firestore
- Material Design
- Git e GitHub

**Destaques:**

- ☁️ Integração com os serviços do Firebase
- 🏗️ Arquitetura organizada seguindo boas práticas do Flutter
- 👥 Desenvolvimento colaborativo utilizando Git, GitHub e estratégia de branching para integração de funcionalidades.
